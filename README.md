# Comunidade Raio-X (RX de Comunidades)

> 🧭 **O Método de Gestão, Curadoria Ativa e Conversão para Grupos de WhatsApp & Mentorias**

Este repositório contém a documentação da metodologia **Comunidade Raio-X**, um sistema prático baseado em 7 passos para mentores e infoprodutores otimizarem seus grupos de alunos no WhatsApp, gerando alto engajamento, reduzindo suporte reativo e criando convergências de negócios para upsell de novos produtos.

## 🚀 O que é a Comunidade Raio-X?

Muitos mentores possuem grupos de WhatsApp cheios, mas frios e desorganizados. O Método Comunidade Raio-X resolve essa dor através de:
1. **Triagem Científica de Logs:** Conversão automática de históricos de conversas em perfis estruturados.
2. **Mapeamento de Sinergias:** Cruzamento de dados de negócios entre os participantes para criar eixos temáticos (sinergias).
3. **Curadoria e Acompanhamento Ativo:** Geração rápida de resumos de encontros síncronas (Zoom) e fixação de atalhos de rotina.
4. **Painel Interativo na Web:** Publicação de um relatório dinâmico (Raio-X) que serve de orientação para a comunidade.

## 📚 Estrutura da Metodologia (Os 7 Passos)

Toda a mentoria está estruturada em passos lógicos de rápida execução:
* **[Passo 1: Persona](docs/01_DIAGNOSTICO_PERSONA/PASSO_01_Persona.md)** — Diagnóstico de quem precisa da otimização.
* **[Passo 2: Jornada do Mentorado](docs/02_JORNADA_MENTORADO/PASSO_02_Jornada.md)** — Do caos e silêncio à clareza estratégica.
* **[Passo 3: Estrutura e Formato](docs/03_ESTRUTURA_FORMATO/PASSO_03_Estrutura.md)** — Modelos de entrega do produto.
* **[Passo 4: Entrega & Acompanhamento](docs/04_ENTREGA_ACOMPANHAMENTO/PASSO_04_Entrega.md)** — Rotinas de curadoria, IA e resumos Zoom.
* **[Passo 5: Precificação](docs/05_PRECIFICACAO/PASSO_05_Precificacao.md)** — Modelos High Ticket e de recorrência mensal.
* **[Passo 6: Oferta & Naming](docs/06_OFERTA_NAMING/PASSO_06_Oferta.md)** — Roteiro de copy e posicionamento.
* **[Passo 7: Canais de Aquisição](docs/07_ESTRATEGIA_VENDAS/PASSO_07_Vendas.md)** — Vendas consultivas e auditorias gratuitas.

## 🛠️ Como rodar a documentação localmente

1. Certifique-se de que possui o Python e pip instalados.
2. Instale as dependências:
   ```bash
   pip install mkdocs mkdocs-material
   ```
3. Inicie o servidor local:
   ```bash
   mkdocs serve
   ```
4. Acesse em seu navegador: `http://127.0.0.1:8000/`
